Aethra Starfire AS4 Flex Printing Machines

The Flex Starfire 25/10PL series is a top-tier industrial flex printer designed for ultra-high-speed and precision printing, ideal for large-format outdoor advertising. It uses Starfire printhead technology with variable droplet sizes of 25PL and 10PL, allowing the user to choose between high-speed bulk printing or ultra-fine image detail.

Technical Specifications

  • Beam Material: Aluminium
  • Max Printing Size: 10.5 ft (3.2 m)
  • Resolution: 360, 480, 720, 1080, 1440 DPI
  • Max Print Speed: 1250 sq.ft/hr (On 2 Pass)
  • Model: Starfire AS4
  • Brand: Aethra
  • Print Heads: Starfire AS4/25/10 PL
  • Bearings: Japanese NSK Bearings
  • Linear System: Japanese THK Linear with 2 Bearing Blocks
  • Main Belt: 25mm Megadyne Italy Belt
  • Pressure System: Negative Pressure System with Auto Pressure
  • Media Sensor: Korean Autonics Sensor
  • Control System: BYHX-A-Type 8 Head
  • Software: Ultraprint RIP Software , Printer Manager
  • Media Heating: 3-Way Digital Control
  • Vacuum Pressure: Manually Adjustable
  • Carriage Height: Precise Carriage Height Adjustment

Supported Media & Features

The AS4 supports various media types including:

  • PVC Banner
  • Flex
  • Vinyl
  • Mesh Banner
  • One-Way Vision
  • Backlit Media
  • Star
  • Blackback

Equipped with auto take-up & feeding, vacuum bed, white skipping, and auto tiling, it is ideal for high-volume production.
